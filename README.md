# Git & GitHub Learning Project

This is my second GitHub project created while learning the fundamentals of Git, GitHub, HTML, and CSS.

## About

This project is a simple webpage that introduces Git and explains its purpose as a version control system. It was built to practice basic web development and Git workflows.

## Features

* Semantic HTML structure
* Responsive viewport setup
* Headings and paragraph content
* Introduction to Git
* Beginner-friendly project

## Technologies Used

* HTML5
* CSS3
* Git
* GitHub

## How to Run

1. Clone the repository:

```bash
git clone https://github.com/your-username/repository-name.git
```

2. Open the project folder.

3. Open `index.html` in your browser.
